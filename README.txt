In order to run:

Part 1 a:

java Receiver1a 8000 <absolute file path>
java Sender1a localhost 8000 <absolute/relative filepath>

Part 1 b:

java Receiver1b 8000 <absolute file path>
java Sender1b localhost 8000 <absolute/relative filepath> <retransmission timeout>

Absolute filepath means for example: /home/vmuser/Desktop/comn1a/test.jpg
